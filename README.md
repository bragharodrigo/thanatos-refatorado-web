# Thanatos Lite - Sistema de Gerenciamento de Cemitérios

# Status do Projeto
Em desenvolvimento (Versão teste finalizada)

# Tecnologias Aplicadas
* Linguagem: Java (Interface Gráfica com Swing)
* Banco de Dados: MySQL
* Conectividade: JDBC (Java Database Connectivity)
* IDE: NetBeans


# Objetivo do Software
O Thanatos Lite é um software desktop projetado para modernizar e informatizar a administração de cemitérios. Seu objetivo principal é garantir a integridade dos dados e agilizar processos operacionais, substituindo controles manuais por um sistema centralizado que gerencia a venda de jazigos, o cadastro de titulares e o registro histórico de sepultamentos.

# Funcionalidades do Sistema
1. Controle de Acesso: Login seguro para funcionários (Admin, Vendedor, Coveiro).
2.  Gestão de Jazigos: Visualização de disponibilidade (Livre/Ocupado) e vinculação de compra.
3.  Cadastro de Titulares: Registro completo de clientes (Nome, CPF, Contato).
4.  Registro de Sepultamentos: Histórico de falecidos vinculados aos jazigos e titulares.
5.  Sistema de Consultas: Busca rápida de titulares, falecidos e situação dos jazigos com opções de edição e exclusão segura.
